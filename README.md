## 🚀 Deploy

### Vercel (recomendado)
1. Crear cuenta en [vercel.com](https://vercel.com)
2. Importar proyecto desde GitHub o drag & drop de la carpeta
3. Framework preset: Astro
4. Deploy

### Netlify
1. Crear cuenta en [netlify.com](https://netlify.com)
2. Drag & drop la carpeta `dist/` después de `pnpm build`

## 🧞 Comandos

| Comando                | Acción                                           |
| :--------------------- | :----------------------------------------------- |
| `pnpm install`         | Instalar dependencias                            |
| `pnpm dev`             | Iniciar servidor local en `localhost:4321`       |
| `pnpm build`           | Compilar sitio a `./dist/`                       |
| `pnpm preview`         | Previsualizar build localmente                   |

## 🛠 Tecnologías

- Astro 6
- Tailwind CSS 4
- Tema oscuro con colores violeta
