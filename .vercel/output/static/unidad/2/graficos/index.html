Tipos de gráficos

📊 Gráfico de barras

Ideal para comparar cantidades entre diferentes categorías. Cada barra representa una categoría y su altura indica el valor.

📈 Gráfico de líneas

Muestra la evolución de una variable a lo largo del tiempo. Útil para ver tendencias y cambios.

🥧 Gráfico circular (torta)

Representa partes de un todo. Cada sector muestra la proporción de cada categoría respecto al total.

📊 Histograma

Similar al de barras pero para datos agrupados en intervalos. Muestra la frecuencia de cada intervalo.
