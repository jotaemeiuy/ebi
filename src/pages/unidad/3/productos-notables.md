<div id="binomio-suma" class="lesson-box">
  <h2>Binomio al Cuadrado (Suma)</h2>
  <p>Fórmula:</p>
  <p class="font-mono text-violet-400">
    (a + b)² = a² + 2ab + b²
  </p>
  <p>Ejemplos:</p>
  <ul class="list-disc pl-5 marker:text-red-500">
    <li>(x + 5)² = x² + 10x + 25</li>
    <li>(2 + y)² = 4 + 4y + y²</li>
    <li>(3a + 7)² = 9a² + 42a + 49</li>
    <li>(m + n)² = m² + 2mn + n²</li>
    <li>(p + q)² = p² + 2pq + q²</li>
  </ul>
</div>

<div id="binomio-diferencia" class="lesson-box">
  <h2>Binomio al Cuadrado (Diferencia)</h2>
  <p>Fórmula:</p>
  <p class="font-mono text-violet-400">
    (a - b)² = a² - 2ab + b²
  </p>
  <p>Ejemplos:</p>
  <ul class="list-disc pl-5 marker:text-red-500">
    <li>(x - 5)² = x² - 10x + 25</li>
    <li>(2 - y)² = 4 - 4y + y²</li>
    <li>(3a - 7)² = 9a² - 42a + 49</li>
    <li>(m - n)² = m² - 2mn + n²</li>
    <li>(p - q)² = p² - 2pq + q²</li>
  </ul>
</div>

<div id="conjugados" class="lesson-box">
  <h2>Binomios Conjugados (Suma por Diferencia)</h2>
  <p>Fórmula:</p>
  <p class="font-mono text-violet-400">
    (a + b)(a - b) = a² - b²
  </p>
  <p>Ejemplos:</p>
  <ul class="list-disc pl-5 marker:text-red-500">
    <li>(x + 5)(x - 5) = x² - 25</li>
    <li>(2 + y)(2 - y) = 4 - y²</li>
    <li>(3a + 7)(3a - 7) = 9a² - 49</li>
    <li>(m + n)(m - n) = m² - n²</li>
    <li>(p + q)(p - q) = p² - q²</li>
  </ul>
</div>

<div id="termino-comun" class="lesson-box">
  <h2>Binomios con Término Común</h2>
  <p>Fórmula:</p>
  <p class="font-mono text-violet-400">
    (a + b)(a + c) = a² + a(b + c) + bc
  </p>
  <p>Ejemplos:</p>
  <ul class="list-disc pl-5 marker:text-red-500">
    <li>(x + 3)(x + 5) = x² + 8x + 15</li>
    <li>(2 + y)(2 + z) = 4 + 2y + 2z + yz = 4 + 2(y+z) + yz</li>
    <li>(a + b)(a + c) = a² + a(b + c) + bc</li>
    <li>(m + 4)(m + 7) = m² + 11m + 28</li>
    <li>(p + 2)(p + 5) = p² + 7p + 10</li>
  </ul>
</div>
